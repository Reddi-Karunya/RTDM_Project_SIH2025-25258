# Real-Time Disaster Training Monitoring System
*A Submission for the Smart India Hackathon 2025 by Team PLEXUS*

---

## Project Overview

This project is a real-time digital platform designed to monitor and evaluate disaster management trainings. The system addresses the need for improved coordination and data-driven insights by integrating two core components: a mobile-first **FlutterFlow application** for live data entry by field officers, and a centralized **Streamlit dashboard** for analytics and visualization by authorities like SDMAs and NGOs. All data is securely stored and managed in real-time using Google Firebase.
